## 4.Django中数据库配置
在项目wms下有个settings.py文件中找到配置项 ***DATABASES***, 配置成MYSQL数据库:

### 4.1.安装pymysql

```shell
pip install pymysql
```

### 4.2.在项目下__init__文件中配置

```python
import pymysql
pymysql.install_as_MySQLdb()
```

### 4.3.配置数据库

```python
# 原来
DATABASES = {
    'default':{
        'ENGINE':'django.db.backends.sqlite3',
        'NAME': os.path.join(BASE_DIR, 'db.sqlite3'),
    }
}

#改成
DATABASES = {
    'default':{
        'ENGINE':'django.db.backends.mysql',
        'NAME': 'wms',
        'USER':'root',
        'PASSWORD':'Love88me',
        'HOST': 'localhost',
        'PORT': '3306',
    }
}
```
